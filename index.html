Glosario de TI

A B C D E F G H I J K L M N - Z

    A

    Algoritmo:

  • Conjunto de instrucciones definidas y ordenadas que permiten realizar una tarea o resolver un problema.

    • API:

    (Interfaz de Programación de Aplicaciones):
  • Conjunto de reglas que permiten la comunicación entre diferentes aplicaciones o servicios.

    • Arquitectura de Software:

  • Estructura base que define cómo interactúan los componentes de un sistema de software

      • B

      Big Data:

    • Conjunto de datos masivos que requieren nuevas tecnologías para su almacenamiento, procesamiento y análisis.

      • Backup:(Copia de seguridad):

    • Proceso de guardar una copia de los datos importantes para recuperarlos en caso de pérdida.

      • Business Intelligence (BI):

    • Tecnologías y estrategias usadas para analizar datos empresariales y tomar decisiones informadas.

        • C

        Cloud Computing (Computación en la nube):

      • Entrega de servicios informáticos (almacenamiento, procesamiento, etc.) a través de internet.

        • Ciberseguridad:

      • Prácticas y tecnologías usadas para proteger sistemas, redes y datos de accesos no autorizados.

        • CRM (Customer Relationship Management):

      • Sistema que gestiona las relaciones e interacciones con clientes actuales y potenciales.

        • D

          D

          Data Center (Centro de datos):

        • Instalación física que alberga sistemas informáticos y componentes asociados, como servidores y almacenamiento.

          • DevOps:

        • Conjunto de prácticas que unifican el desarrollo de software (Dev) y las operaciones de TI (Ops).

          • Dominio:

        • Nombre único que identifica a un sitio web en internet.

          • E

            E

            ERP (Enterprise Resource Planning):

          • Sistema de gestión empresarial que integra y automatiza los procesos clave de una empresa como contabilidad, recursos humanos, logística y producción en una sola plataforma.

            • Ethernet:

          • Tecnología de red utilizada para conectar computadoras dentro de una red local (LAN), que permite la transmisión de datos a alta velocidad mediante cables.

            • Encriptación (Cifrado):

          • Proceso mediante el cual los datos son convertidos en un formato ilegible sin una clave específica, para garantizar su seguridad durante la transmisión o almacenamiento.

            • F

              F

              Firewall (Cortafuegos):

            • Sistema de seguridad que supervisa y controla el tráfico de red, permitiendo o bloqueando comunicaciones según reglas preestablecidas. Es esencial para proteger redes privadas de accesos no autorizados.

              • Firmware:

            • Software embebido en hardware que controla funciones básicas de dispositivos electrónicos como routers, impresoras o teléfonos. Es una combinación de hardware y software.

              • Frontend:

            • Parte visual de una aplicación o sitio web con la que el usuario interactúa directamente. Incluye el diseño, botones, formularios, y todo lo que se muestra en pantalla.

              • G

                G

                Gigabyte (GB):

              • Unidad de medida de almacenamiento digital equivalente a 1,024 megabytes. Se usa comúnmente para describir el tamaño de archivos o la capacidad de discos y memorias.

                • Git:

              • Sistema de control de versiones que permite a los desarrolladores guardar y gestionar cambios en el código fuente de forma colaborativa. Es ampliamente utilizado en proyectos de software.

                • GUI (Graphical User Interface):

              • Interfaz gráfica que permite al usuario interactuar con dispositivos mediante elementos visuales como ventanas, iconos y menús, en lugar de usar solo texto.

                • H

                  H

                  Hardware:

                • Conjunto de componentes físicos que conforman un sistema informático, como la placa base, procesador, memoria RAM, disco duro, teclado, monitor, entre otros. Es la parte tangible del sistema.

                  • HTML (HyperText Markup Language):

                • Lenguaje de marcado utilizado para estructurar el contenido de páginas web. Define la organización de texto, imágenes, enlaces, formularios y otros elementos visuales en la web.

                  • Hosting (Alojamiento web):

                • Servicio que permite publicar un sitio web en internet almacenando sus archivos en servidores que están conectados a la red de forma permanente.

                  • I

                    I

                    IP (Internet Protocol):

                  • Protocolo que asigna direcciones únicas a dispositivos conectados a una red para permitir su identificación y comunicación. Puede ser IP pública o privada, y de tipo IPv4 o IPv6.

                    • IA (Inteligencia Artificial):

                  • Rama de la informática que crea sistemas capaces de simular el comportamiento inteligente humano, como el razonamiento, el aprendizaje, la percepción o la toma de decisiones.

                    • IoT (Internet of Things):

                  • Red de objetos físicos conectados a internet, que recopilan y comparten datos entre sí o con otros sistemas. Ejemplos incluyen relojes inteligentes, termostatos y sensores industriales.

                      • J

                      Java:

                    • Lenguaje de programación orientado a objetos, ampliamente usado para desarrollar aplicaciones empresariales, móviles y de servidor. Su lema “escribe una vez, corre en cualquier lugar” refleja su portabilidad.

                      • JavaScript:

                    • Lenguaje de programación interpretado que se ejecuta en los navegadores web y permite crear sitios web dinámicos e interactivos. No debe confundirse con Java.

                      • JSON (JavaScript Object Notation):

                    • Formato ligero de intercambio de datos, basado en texto, que es fácil de leer para los humanos y de interpretar por las máquinas. Muy usado en APIs y aplicaciones web.

                        • K

                        Kernel:

                      • Núcleo del sistema operativo que gestiona el acceso al hardware, coordina procesos y permite la comunicación entre software y hardware. Es la parte más fundamental del sistema operativo.

                        • Keylogger:

                      • Tipo de malware que registra las pulsaciones de teclas del usuario con el objetivo de obtener información confidencial como contraseñas, datos bancarios o mensajes.

                        • Kubernetes:

                      • Plataforma de código abierto para la automatización del despliegue, escalado y gestión de aplicaciones en contenedores, muy utilizada con Docker.

                          • L

                          LAN (Local Area Network):

                        • Red local que conecta computadoras y dispositivos dentro de un área geográfica limitada, como una oficina, edificio o casa. Permite compartir recursos como impresoras y archivos.

                          • Linux:

                        • Sistema operativo de código abierto basado en Unix, muy utilizado en servidores, sistemas embebidos y entornos académicos por su seguridad, estabilidad y flexibilidad.

                          • Login (Inicio de sesión):

                        • Proceso mediante el cual un usuario se identifica en un sistema mediante credenciales como nombre de usuario y contraseña para acceder a recursos o servicios.

                            • M

                            Malware

                          • Término genérico para referirse a cualquier software malicioso diseñado para infiltrarse, dañar o robar información de sistemas informáticos. Incluye virus, troyanos, spyware, etc.

                            • Machine Learning:

                          • Subcampo de la inteligencia artificial que permite a las computadoras aprender automáticamente a partir de datos, sin necesidad de programación explícita.

                            • Mainframe:

                          • Computadora de gran capacidad utilizada por grandes organizaciones para procesar grandes volúmenes de datos y aplicaciones críticas como sistemas bancarios o gubernamentales.
