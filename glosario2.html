Glosario de TI

N C P Q R S T U V W X Y Z A - N

N

Nube:

  • Tecnología que permite almacenar, gestionar y procesar datos a través de internet en lugar de un disco local. Ejemplo: Google Drive o Dropbox.

    • Nodo:

  • Cualquier dispositivo conectado a una red que puede enviar, recibir o reenviar información. Ejemplo: una computadora, impresora o router.

    • Navegador:

  • Aplicación que permite acceder y visualizar páginas web. Ejemplos comunes son Chrome, Firefox y Safari.

    • O

    Open Source (Código Abierto):

  • Tipo de software cuyo código fuente está disponible para que cualquiera lo use, modifique o distribuya libremente. Ejemplo: el sistema operativo Linux.

    • Oracle:

  • Empresa líder en soluciones de base de datos y software empresarial. Es conocida por Oracle Database, uno de los sistemas de gestión de bases de datos más usados.

    • OAuth:

  • Protocolo de autorización que permite que un sitio o aplicación acceda a información de otro servicio sin compartir la contraseña. Ejemplo: iniciar sesión en Spotify con tu cuenta de Google.

    • P

    Python:

  • Lenguaje de programación popular por su simplicidad y legibilidad. Se usa en desarrollo web, inteligencia artificial, automatización, entre otros.

    • Phishing:

  • Técnica de ciberdelincuencia donde se engaña al usuario para obtener información sensible, como contraseñas o datos bancarios, usualmente mediante correos falsos.

    • Proxy:

  • Servidor intermediario entre el usuario e internet, usado para mejorar seguridad, controlar contenido o mejorar el rendimiento de navegación.

    • Q

    QoS (Quality of Service):

  • Conjunto de tecnologías que aseguran un rendimiento específico (como velocidad o prioridad) para ciertos servicios en redes, muy usado en llamadas VoIP o streaming.

    • Query (Consulta):

  • Instrucción que se utiliza para obtener o modificar información en una base de datos. Ejemplo en SQL: SELECT * FROM usuarios.

    • QR Code:

  • Código de barras bidimensional que puede almacenar datos como URLs. Al escanearse con un móvil, lleva a información o sitios web.

    • R

    Router (Enrutador):

  • Dispositivo que dirige el tráfico entre redes. Es el aparato que usamos en casa para conectar nuestros dispositivos a internet.

    • RAM (Memoria de Acceso Aleatorio):

  • Tipo de memoria volátil que almacena datos temporales mientras se ejecutan programas. A más RAM, más tareas puede manejar el sistema al mismo tiempo.

    • Red:

  • Conjunto de dispositivos interconectados que comparten recursos e información. Puede ser una red doméstica o una red empresarial.

    • S

    SQL (Structured Query Language):

  • Lenguaje usado para gestionar bases de datos relacionales. Se emplea para crear, leer, actualizar o eliminar datos.

    • Servidor:

  • Computadora o sistema que proporciona servicios o recursos a otros dispositivos (clientes). Ejemplo: un servidor web entrega páginas cuando navegas por internet.

    • Software:

  • Conjunto de programas, aplicaciones y datos que hacen funcionar una computadora. Ejemplo: Microsoft Word, Adobe Photoshop.

    • T

    TCP/IP:

  • Conjunto de protocolos que permiten la comunicación entre computadoras en internet. TCP se encarga de asegurar que los datos lleguen completos, e IP se ocupa de la dirección.

    • Token:

  • Elemento que representa credenciales de acceso. Puede ser un dispositivo físico, un archivo digital o un código temporal usado para autenticar al usuario.

    • Trojan (Troyano):

  • Tipo de malware que aparenta ser un software legítimo, pero al ejecutarse da acceso no autorizado a los atacantes.

    • U

    UPS (Sistema de Alimentación Ininterrumpida)

  • Dispositivo que proporciona energía de respaldo a equipos electrónicos durante interrupciones del suministro eléctrico, evitando pérdida de datos y daños en el hardware.

    • Utilización de Recursos:

  • Medida de cuánto de los recursos de un sistema se están utilizando, como CPU, memoria y almacenamiento.

    • USB (Universal Serial Bus):

  • Estándar de conexión para dispositivos periféricos, como teclados, ratones y memorias externas, que permite la transferencia de datos y suministro de energía.

    • V

    VPN (Red Privada Virtual):

  • Conexión segura y cifrada entre dos redes o entre un usuario determinado y una red, permitiendo la transmisión de datos de forma privada a través de internet.

    • VM (Máquina Virtual):

  • Emulación de un sistema informático que permite ejecutar un sistema operativo y aplicaciones como si fuera una computadora independiente.

    • Vishing:

  • Variante del phishing que utiliza llamadas telefónicas para obtener información confidencial, como contraseñas o datos bancarios.

    • W

    Wi-Fi:

  • Tecnología que permite la conexión inalámbrica de dispositivos a una red local, utilizando ondas de radio para la transmisión de datos.

    • Workflow:

  • Serie de tareas y procesos que se realizan para lograr un objetivo específico, a menudo automatizados mediante software.

    • WAF (Web Application Firewall):

  • Firewall diseñado específicamente para proteger aplicaciones web, filtrando y monitoreando el tráfico HTTP.

    • X

    XML (Lenguaje de Marcado Extensible):

  • Lenguaje de marcado utilizado para definir reglas de formato de documentos, permitiendo la estructuración y el transporte de datos de manera legible tanto para humanos como para máquinas.

    • XaaS (Todo como Servicio):

  • Categoría de servicios en la nube que incluye IaaS, PaaS, SaaS y más, proporcionando recursos y aplicaciones a través de internet.

    • XSS (Cross-Site Scripting):

  • Vulnerabilidad de seguridad en aplicaciones web que permite a atacantes inyectar scripts maliciosos en páginas vistas por otros usuarios.

    • Y

    Yottabyte:

  • Unidad de medida de almacenamiento equivalente a 1 billón de terabytes, utilizada para describir grandes volúmenes de datos.

    • YAML (YAML Ain’t Markup Language):

  • Lenguaje de serialización de datos legible por humanos, utilizado para representar estructuras de datos complejas de manera sencilla.

    • Yara:

  • Herramienta de detección de malware basada en reglas, utilizada para identificar y clasificar amenazas informáticas.

    • Z

    Zabbix:

  • Es un sistema de monitorizacion de redes creado por Alexei Vladishev. Esta diseñado para monitorizar y registrar el estado de varios servicios de red, servidores, y hardware de red

    • Zigbee:

  • Zigbee es un protocolo de comunicacion inalámbrica de bajo consumo que se utiliza para crear redes de malla (mesh networks) entre dispositivos de Iot. Permite que dispositivos como luces, enchufes inteligentes, sensores y otros dispositivos domoticos se comuniquen entre si, incluso sin depender de Wi-Fi o bluetooth.

    • Zettabyte:

  • Un Zettabyte (ZB) es una unidad de almacenamiento de información digital que equivalea mil exabytes, o un billon de terabytes. Es un unidad muy grande, utilizada para expresar cantidades masivas de datos digitales, como el volumen total de datos generados en internet a nivel mundial.
